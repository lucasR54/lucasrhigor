Exercício 1

Esse programa pede ao usuário (você) para inserir um número, em seguida ele vai verificar se o número é positivo, negativo ou neutro.

Exercício 2

Esse programa pede ao usuário (você) sua idade, assim ele vai dizer se você é criança (0-12 anos) adolescente (13-17 anos), adulto (18-64 anos)
ou idoso (65 anos ou mais).

Exercício 3

Esse programa pede ao usuário (você) para inserir um número correspondente a um dia da semana (1 para Domingo, 2 para Segunda, etc.).
Dessa maneira ele responde qual é o dia da semana a partir com base nesta ordem.

Exercício 4

O programa imprime números de 1 a 10 utilizando um laço for. O usuário (você) não precisa inserir nenhum valor.

Exercício 5

Esse programa pede ao usuário (você) para inserir números inteiros repetidamente até que insira o número zero.
Então O programa vai exibir a soma de todos os números inseridos antes do zero.

Exercício 6

Esse programa pede ao usuário (você) para adivinhar um número surpresa entre 1 e 10. Ele vai continuar pedindo
para tentar novamente até que você acerte.

Exercício 7

Esse programa que pede ao usuário (você) para inserir 5 números inteiros, em seguida ele vai armazenar esses valores e exibir o maior e menor número inserido.

Exercício 8

Esse programa pede ao usuário (você) para inserir uma quantidade de números, que você desejar, para a realização de uma média e no final ele exibe o resultado.
O programa calcula isto com o uso de vetores.

Exercício 9

Esse programa pede ao usuário (você) para inserir uma quantidade de números, que você desejar, para depois selecionar um
valor específico desta ordem digitada e apresentar os valores que são maiores que o número determinado. O programa também utiliza vetores aqui.

Exercício 10

Neste último programa, ele pede ao usuário (você) para inserir uma quantidade de palavras, quantas você desejar,
para em seguida selecionar um valor específico desta ordem digitada e apresentar a quantidade de vezes que a palavra é repetida.
