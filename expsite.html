O conceito do Site

Esse site se trata de um Portifolio que abrange uma lista de exercícios, mas no geral é um trabalho interdiciplinar do 1DSB que
junta as seguintes materias do curso de DS: Programação Web(PW), Técnicas de Programação e Algoritmo(TPA) e Desin Digital(DD).
O trabalho era para realizado em dupla para o desenvolvimento da lista em JavaScript, como foi discutido em sala.

Para finalizar, é importante constar que este se trata de um TRABALHO ACADÊMICO SEM FINS LUCRATIVOS
